# ShopEasy Marketing Analytics – Final Project

## Project Overview

This project analyzes ShopEasy customer, marketing, and sales data to identify the causes of declining customer engagement, low conversion rates, and reduced marketing ROI.

The analysis was conducted using SQL, Power BI, and Python to support data-driven business decision-making.
